# instagramTrend
```
爬取 instagram 的 hashtag 之相關文章內容、相關標簽
```

## 專案安裝
```
git clone https://github.com/welch79328/instagramTrend.git
```

```
pip install -r requirements.txt
```

## 爬取賽事
```
python execution.py -t hashtag -hl 100 -al 100

-t => 爬取類型
-hl => 標籤數量
-al => 文章數量
```

## 資料庫結構
```
instagramTrend.sql
```
